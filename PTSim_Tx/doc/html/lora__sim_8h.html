◆ _receiveGPS()

void _receiveGPS ( void *  port,
char *  str_coord 
)

Lis le port jusqu'à recevoir les coordonnées du tracker.

Paramètres
*portPort de communication avec le tracker
*str_coordChaîne de caractère où mettre les coordonnées ("lat;lon")

Lis le port jusqu'à recevoir un message contenant "GPS:" signifiant que le tracker envoie ses coordonnées. Si le message ne contient pas "GPS:", la commande "GetGPS" est renvoyée.
